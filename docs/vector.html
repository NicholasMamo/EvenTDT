2. Vector Space Model (VSM)¶

Most document representation occurs in the Vector Space Model (VSM). EvenTDT provides classes to help you work with the VSM. Keep reading to learn more about the different VSM classes available in EvenTDT.

Vectors¶

The vsm.vector.Vector class is used in clustering and other applications that adopt the Vector Space Model.

class eventdt.vsm.vector.Vector(dimensions=None, *args, **kwargs)¶

The vsm.vector.Vector class is the smallest building block in the Vector Space Model. It is used for tasks such as clustering and to represent documents. Vectors are based on objects.Attributable so that they may have additional properties.
